#get_route_origin_destination(destination:, origin:, avoid: nil, connections: nil, flag: "shortest", headers: {}, params: {}) ⇒ Object

Get the systems between origin and destination.

This endpoint is cached for up to 86400 seconds.

Parameters:

  • destination (Integer) —

    destination solar system ID

  • origin (Integer) —

    origin solar system ID

  • avoid (Array) (defaults to: nil) —

    avoid solar system ID(s)

  • connections (Array) (defaults to: nil) —

    connected solar system pairs

  • flag (String) (defaults to: "shortest") —

    route security preference. Must be one of: shortest, secure, insecure

  • params (Hash) (defaults to: {}) —

    Additional query string parameters

  • headers (Hash) (defaults to: {}) —

    Additional headers

def get_route_origin_destination_raw(destination:, origin:, avoid: nil, connections: nil, flag: "shortest", headers: {}, params: {})
  params.merge!("avoid" => avoid, "connections" => connections, "flag" => flag)
  get("/route/#{origin}/#{destination}/", headers: headers, params: params)
end
